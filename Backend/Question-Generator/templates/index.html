AI Grading Engine

How Auto-Grading Works

Our multi-layer grading engine handles every question type intelligently — from instant MCQ checking to nuanced LLM-powered evaluation of essays and code.

Student Submits Answer

Responses are securely captured and sent for evaluation immediately on submission.

Rule-Based Instant Grading

MCQ and True/False questions are graded instantly with exact matching. Multi-select uses partial credit logic.

LLM Semantic Evaluation

Short and long answers are evaluated by a 70B LLM against the expected answer and rubric. F1 overlap acts as a fallback.

Code Execution & Sandbox

Code questions are run in a sandboxed environment. Output is compared against test cases with detailed feedback.

Results & Feedback Delivered

Scores, per-question feedback and explanations display instantly on the student's results page.
